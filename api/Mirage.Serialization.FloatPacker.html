    Remarks

    Positive and Negative values need to be unpacked differnely so that they both keep same precision

    Example 10 bits (max uint = 1023):
    p = precision

    Positive values have uint range 0 to 511
    Unpacked: 0 to max

    Negative values have uint range 512 to 1023
    Unpacked using same as positive: max+p to max*2+p
    but we want range -max to -p
    so we need to subtrack -1024 so range is -512 to -1
    then scale to unpack to -max to -p
